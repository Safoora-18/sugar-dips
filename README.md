## рџљЂ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Safoora-18/sugar-dips.git
   ```

2. **Move the project into your XAMPP htdocs folder**
   ```
   xampp/htdocs/
   ```

3. **Start Apache and MySQL** in the XAMPP control panel

4. **Import the MySQL database** (see `/database` for the SQL file)

5. **Configure Firebase Authentication** with your project credentials

6. **Configure Razorpay API keys** in the payment integration files

7. **Launch the app**
   ```
   http://127.0.0.1/sugardips/
   ```
